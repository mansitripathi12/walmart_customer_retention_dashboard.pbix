# Retail Customer Retention Analytics – Walmart

## Project Overview

This project focuses on analyzing retail customer data to understand customer retention patterns and purchasing behavior at Walmart. Customer retention is a critical factor in the retail industry, as retaining existing customers is often more cost-effective than acquiring new ones. By analyzing customer transaction data, this project aims to identify key factors that influence repeat purchases and long-term customer loyalty.

## Objective

The main objective of this project is to explore customer data and identify trends related to customer retention. The analysis helps in understanding how frequently customers return to make purchases, which customer segments contribute the most to sales, and what factors influence customer loyalty.

## Tools Used

* Microsoft Power BI
* Data Visualization
* Data Analysis

## Key Analysis

* Customer segmentation based on purchasing behavior
* Identification of repeat customers and retention patterns
* Sales trends and customer purchase frequency
* Interactive dashboard visualization for better insights

## Outcome

Using Power BI, an interactive dashboard was created to visualize important metrics related to customer retention and purchasing behavior. The dashboard helps in identifying patterns in customer engagement and provides insights that can help retail businesses improve marketing strategies, enhance customer experience, and increase long-term customer loyalty.
